# 🚲 快趣出行 · 附近车辆查询工具 使用教程

> 本工具支持自定义地点，无需区分校区，只需设置好经纬度即可查询附近车辆信息。

---

## 📍 快速开始

### 第一步：添加你的常用地点

使用 `add` 指令添加地点，格式如下：

```
add [名称] [经度] [纬度]
```

> **💡 提示：** 经纬度请精确到小数点后 **8 位**，否则定位不够精确。  
> 可前往 👉 [https://jingweidu.bmcx.com](https://jingweidu.bmcx.com) 查询精确经纬度。

**示例：**

```
add 学生公寓7栋 116.03523691 28.68901457
add 图书馆正门 116.03612345 28.69012345
add 食堂北门   116.03411111 28.68755555
```

---

### 第二步：查看已添加的地点列表

```
ls
```

**示例输出：**

```
ID   名称            经度            纬度
------------------------------------------
1    学生公寓7栋     116.03523691    28.68901457
2    图书馆正门      116.03612345    28.69012345
3    食堂北门        116.03411111    28.68755555
```

---

### 第三步：选择地点并查询附近车辆

直接输入地点对应的 **数字 ID** 即可查询：

```
[地点数字ID]
```

**示例：**

```
1
```

> 输入 `1` 后，系统将自动查询**学生公寓7栋**附近的可用车辆信息。

---

## 🗑️ 删除地点

如需删除某个地点，使用 `del` 指令：

```
del [地点数字ID]
```

**示例：**

```
del 2
```

> 以上示例将删除 ID 为 `2` 的地点（图书馆正门）。

---

## ❓ 帮助指令

忘记指令了？随时输入 `help` 查看所有可用指令：

```
help
```

---

## 📋 指令速查表

| 指令     | 格式                       | 说明                   |
| -------- | -------------------------- | ---------------------- |
| 添加地点 | `add [名称] [经度] [纬度]` | 添加一个新的常用地点   |
| 删除地点 | `del [ID]`                 | 按 ID 删除已保存的地点 |
| 查看列表 | `ls`                       | 显示所有已保存的地点   |
| 选择地点 | `[ID]`                     | 查询该地点附近的车辆   |
| 帮助     | `help`                     | 显示帮助信息           |

---

## 🔍 如何查询精确经纬度

1. 打开 [https://jingweidu.bmcx.com](https://jingweidu.bmcx.com)
2. 在地图上定位到你想要查询的地点
3. 复制页面显示的经纬度（保留至少 **8 位小数**）
4. 使用 `add` 指令将其添加到工具中

---

> ⚠️ **注意事项**
>
> - 经纬度精度不足会导致查询范围偏移，建议保留 **8 位小数**
> - 所有快趣出行车辆使用统一 API，**无需区分校区**，全平台通用
